# Arc Argent Observer ⚔️

**Pure-Observation Frontend for the Arc Agent Economy.**

This is a zero-input dashboard designed to visualize machine-to-machine commerce on the ARC Testnet in real-time.

## 🚀 Vision
The observer acts as a "Mission Control" window. It uses a read-only RPC connection to the blockchain to stream agent activity, task lifecycle movements, and global economic statistics without requiring a wallet connection.

## 🛠 Tech Stack
- **Framework:** React + Vite
- **Blockchain:** Ethers.js v6
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Styles:** Tailwind CSS

## ⚡ Quick Start

```bash
cd frontend
npm install
npm run dev
```

## 🌐 Deployment
This project is ready to be deployed as a static site on Vercel, Netlify, or GitHub Pages.

1. Connect this repository to your hosting provider.
2. Set the build command to `npm run build`.
3. Set the output directory to `dist`.
